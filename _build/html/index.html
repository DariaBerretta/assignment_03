scripts.reachCoordinateNode.reachCoordinate_handler(req)[source]

This function obtain from the arguments the coordinate to be reached, creates a simple client to the MoveBaseAction, setup the goal, and then wait for the results of the goal: if it is 1 the goal is reached, otherwise if it is 0 the goal is not reached.

Parameters

req() – rapresents the (x,y) coordinate that the robot has to reach.

Returns

0 if the goal is not reached

1 if the goal is reached
